3.5 Lab: Explorer's Blog

In this lab you'll put yourself in the shoes of an interstellar explorer writing logs from their adventures around the galaxy. You'll create a multi-page blog from scratch using 11ty's templating and layout features. Your site should utilize Liquid templates and layouts so all pages share the same HTML header content. Additionally, each blog post will be written in Markdown and transformed to HTML at built time using 11ty.

Lab Objectives

  1. Create a new 11ty site utilizing templates and layouts.
  2. Create a shortcode in 11ty to build reusable content blocks that can be used across your site.
  3. Use Markdown to create easily-written content for pages and articles within a statically-generated site.

Client Design Notes

Note: this week's lab doesn't not have an associated wireframe. 

Lab Requirements

  1. Create a new base 11ty site starting from Jeff's GitHub 11ty Boilerplate. Make sure you create a new repository based on the template repo (see first video): https://github.com/jsiarto/11ty-boilerplate
  2. Create two blog posts using Markdown focused on the subject of exploring distant worlds and stars and in our galaxy.
  3. The final site must have two blog post pages, a homepage which includes a short blurb and link to the full blog posts, and an about page (4 total HTML pages).
  4. The final site must make use of at least one 11ty layout file for the main HTML headers and footers, and one shortcode for reusable content.
  5. Your site should include at least 10 CSS rulesets.
  6. Your site should have a consistent color palette, look and feel across all pages.
  7. Your site should be published to Netlify and public links from GitHub and Netlify should be submitted to D2L.
  8. Submit your public URLs to the 3.5 Lab Assignment dropbox.
